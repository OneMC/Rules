# DEV Skill - 开发文档编写

记录开发过程中的实现要点和问题解决方案。

---

## 执行流程

### 触发条件
- 代码行数 > 50 行
- 涉及 2 个以上模块交互
- 遇到需要记录的问题和解决方案

### 阶段 1：实现要点记录

**输出** `dev-draft.md`：
- 实现思路简述
- 关键代码片段
- 注意事项

### 阶段 2：问题与解决

**输出** `dev-draft.md`（重点）：

每个问题包含：
- **现象**：发生了什么
- **原因**：根本原因分析
- **解决**：具体解决方案（含代码）
- **预防**：如何避免再次发生

### 阶段 3：正式输出

**执行**：
1. 调用 doc-management skill，创建/更新 `{功能名}_dev.md`
2. 确保 frontmatter 完整
3. 关联到对应的 PRD 和 ARC
